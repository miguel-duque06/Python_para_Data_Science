# 🧠 Fundamentos de Data Science com Python

> Aprender dados na prática — um notebook de cada vez.

Este repositório reúne uma coleção de notebooks desenvolvidos ao longo dos meus estudos em **Ciência de Dados**, com foco em **Python** e nas principais etapas do trabalho com dados.

Aqui você não vai encontrar apenas código — mas um registro da minha evolução aprendendo a **explorar, transformar, analisar e visualizar dados**.

---

## 📌 Sobre este repositório

Cada notebook representa um passo importante dentro do fluxo de trabalho em Data Science.

A ideia é simples:
**aprender fazendo — e documentar o processo.**

---

## 📂 O que você vai encontrar aqui

### 🔍 Conhecendo os dados

* `conhecendo_biblioteca.ipynb`
  Primeiro contato com dados e bibliotecas essenciais.

---

### 🧹 Limpeza e tratamento

* `limpeza_tratamento_churn.ipynb`
  Preparação de dados, tratamento de inconsistências e valores ausentes.

---

### 🔄 Transformação de dados

* `transformacao_e_manipulacao_dados.ipynb`
  Reestruturação e manipulação de dados para análise.

---

### 📊 Seleção e agrupamento

* `pandas_selecao_e_agrupamento.ipynb`
  Filtragem, agrupamento e extração de informações relevantes.

---

### 📈 Visualização de dados

* `data_vizualization.ipynb`
  Criação de gráficos para interpretação e comunicação de dados.

## 🧪 Como pensar este repositório

Os notebooks seguem, de forma geral, o fluxo real de um projeto de dados:

1. Entender a base
2. Limpar e tratar os dados
3. Transformar e organizar
4. Analisar e agrupar
5. Visualizar e gerar insights
